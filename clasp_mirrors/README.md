# Clasp mirrors (`clasp_mirrors/`)

This tree is generated by **`scripts/clone_clasp_mirrors.mjs`**: one subdirectory per **Apps Script `scriptId`**, cloned with **`clasp clone`** / refreshed with **`clasp pull`**.

**Git:** Only **safe** artifacts belong in the remote repo: `README.md`, **`MIGRATION_CHECKLIST.tsv`**, **`MANIFEST.json`**, and **`.clasp.json`** per project (maps folder → `scriptId`). **`*.js`** and **`appsscript.json`** under each mirror are **gitignored** — script files contain secrets; `appsscript.json` is recreated by clasp. After `git clone`, run **`clasp pull`** inside each `clasp_mirrors/<scriptId>/` (or run `node scripts/clone_clasp_mirrors.mjs`) to populate local files.

**Canonical copy on your machine:** production Apps Script code should be maintained **here** (`clasp_mirrors/<scriptId>/`), with **`clasp push`** / **`clasp pull`** from that directory. The thematic **`google_app_scripts/**`** tree in the repo is **reference only** (readable layout, docs).

## Regenerate

```bash
cd /path/to/tokenomics
node scripts/clone_clasp_mirrors.mjs
```

- **`MIGRATION_CHECKLIST.tsv`** — each row is a `google_app_scripts/**/*.gs` file: basename match, content-hash match against mirrored files, embedding in `Code.js`, or `UNMAPPED`.
- **`MANIFEST.json`** — last run metadata and clone/pull errors.

Options: `--dry-run`, `--skip-clone` (checklist only), `--force-clone` (delete `<scriptId>/` and re-clone — **destructive**).

## Workflow

1. Treat **`clasp_mirrors/<scriptId>/`** as the folder clasp expects for that cloud project (**master copy** for deploying to Google).
2. Resolve `UNMAPPED` / `BASENAME_ONLY` rows in the TSV by moving or merging reference `.gs` into the right mirror folder (or note that the cloud uses **`Code.js`** while git uses another name — compare contents).
3. **`cd clasp_mirrors/<scriptId>`** then **`clasp push`** when that project’s mirror is ready; optionally copy or reconcile changes into **`google_app_scripts/**`** for documentation only.

## Security

- **Rotate** any token that ever appeared in a committed `Code.js` (or other mirror `.js`): BotFather Telegram tokens, JWTs, generic high-entropy keys — treat them as compromised if they hit GitHub, even after removal from the latest commit.
- Prefer **Script Properties** / env-style config in Apps Script instead of literals in source.
- Do not commit **`clasp_mirrors/**/*.js`** or **`clasp_mirrors/**/appsscript.json`** (enforced in root `.gitignore`).
